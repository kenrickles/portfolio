Project Description

Tasklion seeks to help everyone who requires tasks to be completed to find suitable taskers to help them complete the task

Technologies Used

Frontend: React.js, Javascript

Backend: Node.js, Express, Sequelize, PostgresSQL

Deployment: Heroku

Prototype Features

  • The user will be able to input, edit his or her input
  • The user will be able to select a tasker based on pricing and tasker's information after keying in required details
  • The user will be able to see the progress bar
  • The user will be able to see a confirmation page
