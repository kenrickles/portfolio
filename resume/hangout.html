Project Description

Hangout? seeks to connect new people with common interests and save money while doing the group activities together

Technologies Used

We decided to separate the frontend of the app and the backend separately.

Frontend: React.js, Javascript, React Router

Backend: Node.js, Express, Sequelize, Postgres (For storing the user information, Firebase Real-time database (For messaging)

Deployment: Heroku

Prototype Features

  • The creator of an activity (user) can create, edit, delete an activity
  • The participant of an actvitiy (user) can join and leave an activity
  • Users automatically joins the group chat when they join the activity
  • Users can chat with one another with a real-time chat function

Key Learning Points

  1. Leveraged on advanced react frameworks such as useContext and useReducer to manage and share states across multiple components
  2. Learnt to work with 2 repository, one for the front end and the other for the backend application
  3. Implemeneted a real-time messaging system using Firebase Real-Time Database (NoSql Database)
  4. Implemented deeplinking with react-router
  5. Setting cookies across to the frontend from the backend (crossorigin)
